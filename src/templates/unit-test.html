Stub and Mock

  • Stub - an object that provides predefined answers to method calls;
  • Mock - an object on which you set expectations;
  • Fake - an object with limited capabilities (for the purposes of testing), e.g. a fake web service.

Test runners

  • A test runner is a Ttst-framework agnostic.
  • It allows running tests on multiple browsers quickly.
  • It integrates with continuous builds systems.
  • Some examples of "test runners" are Testem and Testacular (Karma)
