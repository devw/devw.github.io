Setup git for windows

  • Git is a distributed revision control system
  • Download and install "git bash" for windows.
  • http://git-scm.com/download/win

Generating SSH keys

  • Windows OS: 
        > # windows
    > ssh-keygen.exe
    > cat .ssh/id_rsa.pub | clip
  • OSX users: 
        > # osx
    > ssh-keygen
    > more .ssh/id_rsa.pub | pbcopy
  • Add your SSH key to your GitHub account.

Change the URI (URL) for a remote Git repository

  • Manage the set of repositories ("remotes") whose branches you track.
  • View existing remotes: 
        > git remote -v
    # origin  https://github.com/user/repo.git (fetch)
    # origin  https://github.com/user/repo.git (push)
  • Change the 'origin' remote's URL: 
        > git remote set-url \
    origin ssh://git@github.com/user/repo.git


    > git remote -v
    # Verify new remote URL
    # origin  ssh://git@github.com/user/repo.git (fetch)
    # origin  ssh://git@github.com/user/repo.git (push)

Git commands

  • Removing multiple files from a Git repository that have already been deleted from disk: 
        git rm $(git ls-files --deleted)

Node.js

  • Node.js is an open source, cross-platform runtime environment for server-side and networking applications
  • npm is a package manager for JavaScript, and is the default for Node.js
  • npm is bundled and installed automatically with Node.js

Create a package.json file

  • This will ask you a bunch of questions, and then write a package.json for you: 
        > mkdir 01app
    > cd 01app
    > mkdir app
    > mkdri test

    # Create a package.json file
    > npm init

Install packages

  • This command installs a package, and any packages that it depends on. 
        npm install karma --save-dev
    npm install karma-jasmine --save-dev
    npm install karma-chrome-launcher --save-dev
