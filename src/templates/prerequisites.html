Setup git for windows

  • Git is a distributed revision control system
  • Download and install "git bash" for windows.
  • http://git-scm.com/download/win
  • How do you copy and paste into Git Bash? If the "QuickEdit Mode" is on, "right clicking" and press "Insert" to copy and past.

Generating SSH keys

  • Windows OS: 
        > # windows
    > ssh-keygen.exe
    > cat .ssh/id_rsa.pub | clip
  • OSX users: 
        > # osx
    > ssh-keygen
    > more .ssh/id_rsa.pub | pbcopy
  • Add your SSH key to your GitHub account.

Change the URI (URL) for a remote Git repository

  • git remote manages the set of repositories ("remotes") whose branches you track.
  • View existing remotes: 
        > git remote -v
    # origin  https://github.com/user/repo.git (fetch)
    # origin  https://github.com/user/repo.git (push)
  • Change the 'origin' remote's URL: 
        > git remote set-url \
    origin ssh://git@github.com/user/repo.git

    > git remote -v
    # Verify new remote URL
    # origin  ssh://git@github.com/user/repo.git (fetch)
    # origin  ssh://git@github.com/user/repo.git (push)

Git commands

  • Removing multiple files from a Git repository that have already been deleted from disk: 
    
    git rm $(git ls-files --deleted)
  • Undo a local commit 
        git reset --soft HEAD~1
  • Undo a local add 
        git reset .
  • You want to switch branches, but you don’t want to commit what you’ve been working on yet. 
    
    > git stash
    > git stash list
    # stash@{0}: WIP on master: 049d078 added the index file
    # stash@{1}: WIP on master: c264051 Revert "added file_size"

    > git stash apply stash@{1}

Node.js

  • Node.js is an open source, cross-platform runtime environment for server-side and networking applications
  • npm is a package manager for JavaScript, and is the default for Node.js
  • npm is bundled and installed automatically with Node.js

Create a package.json file

  • This will ask you a bunch of questions, and then write a package.json for you: 
    
    > mkdir 01app # Create a directory named 01app
    > cd 01app # change directory
    > mkdir app # Create a directory named app
    > mkdri test # Create a directory named test

    > npm init # Create a package.json file

Install packages

  • This command installs a package, and any packages that it depends on. 
    
    > npm install karma --save
    > npm install karma-jasmine --save
    > npm install karma-chrome-launcher --save

    > ls node_modules/
    # karma karma-jasmine karma-chrome-launcher

Add a path to PATH

  • PATH is an environmental variable in Linux and other Unix-like operating systems that tells the shell which directories to search for executable files
  • 
    > export PATH=node_modules/karma/bin:$PATH

Bower

  • Bower is a package manager for the web.
  • Install it with npm: 
    
    npm install -g bower
  • Show the help message 
    
    bower -h

Grunt

  • Grunt is a JavaScript task runner;
  • It has a robust ecosystem of plugins focused around building modern web applications;
  • It includes tools such as integrated testing, minification and LiveReload;
  • It is built atop Node.js, needed in order to run.

Gruntfile.js

  • Watches files for changes and runs tasks based on the changed files: 
    
    ...
    styles: {
        files: ['<%= config.app %>/styles/{,*/}*.css'],
        tasks: ['newer:copy:styles', 'autoprefixer']
    },
    ...

grunt.registerTask
